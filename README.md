# Tarea-4.1

# Captura de Datos de Participante

## Descripción

Aplicación desarrollada en Java Swing utilizando NetBeans que permite capturar y presentar la información de un participante.

## Datos capturados

- Matrícula
- Nombres
- Apellidos
- Carrera

## Funcionalidades

- Capturar los datos ingresados por el usuario.
- Presentar los datos mediante un cuadro de diálogo.
- Aplicación de Programación Orientada a Objetos mediante una clase independiente para el procesamiento de los datos.

## Tecnologías utilizadas

- Java
- Java Swing
- Apache NetBeans

## Estructura del proyecto

```
src/

ProcesarDatos.java
FormularioGUI.java
Main.java
```

## Cómo ejecutar

1. Abrir el proyecto en NetBeans.
2. Ejecutar la clase Main.java.
3. Completar los datos.
4. Presionar "Capturar".
5. Presionar "Presentar".
